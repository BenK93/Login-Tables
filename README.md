# React LoginPanel & Tables with Redux

# Getting Started 

1) Clone repo.
2) Change dir to repo (cd <Login-Tables>).
3) Run :
### `npm install`
4) Run :.
### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
